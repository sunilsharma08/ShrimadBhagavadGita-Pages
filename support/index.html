Using Gitansh
Gitansh works using on-device Apple Intelligence and can work offline once available.

Available only if:
• Device supports Apple Intelligence
• iOS 26 or above
• Apple Intelligence is enabled in settings
Common Issues

• Gitansh not visible → Device not supported or iOS below 26
• Gitansh not working → Enable Apple Intelligence in settings
• Content not loading → Check internet connection
• App issues → Restart or update the app
